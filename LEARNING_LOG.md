## Day 1 — 2026-05-22

**Concepts covered:**
- Shebang line (`#!/bin/bash`) — tells the OS which interpreter to use
- Variables — assign with `=` (no spaces), access with `$`
- `echo` — print to the terminal
- Double quotes around `"$var"` — protects values that contain spaces

**Key insight:**
- Without `$` you're naming the box. With `$` you're opening it.
- `echo quote` prints the word "quote". `echo $quote` prints the value inside it.
